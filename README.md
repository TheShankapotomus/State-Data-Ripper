
## State Data Ripper


This small script is used to screenscrape Massachusetts congressional data from the state sites. The output is in CSV format and the input file handles both CSV formatting and the site links.

* Python 3.4 or above required

* BeautifulSoup4 required

BS4 Install instructions:

	Windows: python -m pip install beautifulsoup4
	Linux:   pip3.4 install beautifulsoup4
